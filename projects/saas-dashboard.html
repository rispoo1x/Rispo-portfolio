What it does

A production-grade SaaS analytics dashboard built on Next.js App Router with Supabase as the backend. The entire data layer is a custom REST API — no abstraction-heavy ORM magic, just clean SQL and typed responses.

The dashboard supports multiple user roles: admins see everything, team members see only their data, and guests get read-only snapshots. Auth is handled through Supabase Auth with Row Level Security policies enforced at the database level — not just in the frontend.

Charts update in real time using Supabase Realtime subscriptions. No polling, no full-page reloads — data streams directly into the Recharts components as it lands in the database.

📡

Real-time data streaming

Supabase Realtime pushes database changes directly to the UI. Charts update live without a single network request.

🔐

Row-level security

Data access is enforced at the Postgres level with RLS policies — not just UI conditionals. No privilege escalation possible.

⚙️

Custom REST API

Every endpoint is hand-built with Next.js API routes: typed request/response, pagination, filtering, and rate limits.

📱

Fully responsive

The dashboard adapts to any screen — charts reflow, tables collapse to cards, and sidebars become bottom sheets on mobile.
